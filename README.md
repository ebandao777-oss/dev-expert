# 编程专家 - dev-expert

面向软件工程师和开发团队的编程全生命周期助手。覆盖从需求分析、代码生成、Bug诊断到重构、测试、文档的全链路，通过12个子技能提供专业化支持。

## 子技能列表

| 子技能 | 功能 | 触发关键词 |
|--------|------|-----------|
| API设计 | 根据业务需求设计RESTful或GraphQL API接口 | API设计, RESTful, GraphQL, 接口规范 |
| Bug诊断 | 分析错误日志和异常堆栈，定位Bug根因并给出修复方案 | Bug诊断, debug, 报错, 异常堆栈 |
| Karpathy编码规范 | 提供Karpathy核心编码哲学：思考优先、简洁至上 | Karpathy编码规范, Karpathy, 编码哲学 |
| Spec驱动开发 | 编码前对齐需求规格，使用OpenSpec artifact flow | Spec驱动开发, spec, 需求对齐 |
| 代码审查 | 审查代码质量，发现Bug、安全漏洞和代码异味 | 代码审查, code review, 代码异味 |
| 代码生成 | 根据功能需求生成高质量代码，含错误处理和边界条件 | 代码生成, 生成代码, 实现功能 |
| 任务拆解与执行 | 将复杂需求拆分为原子任务，按Wave分组执行 | 任务拆解与执行, 任务分解, Wave执行 |
| 技术选型 | 根据项目需求推荐合适技术栈、框架和工具 | 技术选型, 技术栈, 框架选型 |
| 文档生成 | 根据代码生成技术文档、README和API文档 | 文档生成, API文档, README |
| 测试用例生成 | 生成单元测试、集成测试，覆盖正常和异常路径 | 测试用例生成, 单元测试, 测试用例 |
| 重构建议 | 分析代码结构，识别坏味道并提供重构方案 | 重构建议, 重构, 坏味道 |
| 项目记忆管理 | 捕获上下文和决策，实现跨会话项目记忆沉淀 | 项目记忆管理, 项目记忆, 跨会话 |

## 使用方法

通过 Marvis 对话自然触发，说出需求即可自动匹配对应子技能。

## 协同技能

编程专家为独立技能套件，暂无可直接联动的其他职业技能。如需在编程任务中集成外部数据或服务，可使用主Agent的web_search/web_fetch工具获取。
